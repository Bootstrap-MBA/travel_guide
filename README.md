## Getting started

0. Install the deps: 

  run `yarn`.

1. Run postcss to compile tailwind:

  `yarn run postcss css/tailwind.css -o css/compiled.css`

2. Tell jekyll to serve the site:

  `jekyll serve`


## Deploying
